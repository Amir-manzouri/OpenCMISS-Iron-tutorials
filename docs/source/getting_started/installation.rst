Installation
============

This section describes how to install OpenCMISS. Two options are available:

.. toctree::
  :numbered:
  :maxdepth: 1

  installation/docker
  installation/source

Once installed, the OpenCMISS-Iron python module will be available from python.

.. Important::

    If you are new to OpenCMISS, we recommend using the OpenCMISS-Iron Dockers, and following the instructions for running JupyterLab.
